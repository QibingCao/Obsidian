# 解答
```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        if (root == null || root == p || root == q) return root; //如果等于p或q直接返回
        
        TreeNode left = lowestCommonAncestor(root.left, p, q); 
        TreeNode right = lowestCommonAncestor(root.right, p, q);
        if (left == null) return right; //如果左子节点均无返回又节点
        if (right == null) return left; //如果右子节点均无返回左节点
        return root; // 如果左右子节点均有则返回当前值
    }
}

```
## DFS or 后序遍历
```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    TreeNode ans = null;
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        dfs(root, p, q);
        return ans;
    }

    public boolean dfs(TreeNode root, TreeNode p, TreeNode q){
        if(root == null){
            return false;
        }
        //在当前节点
        boolean inCurrentNode = root.val == p.val || root.val == q.val;
        //在左节点
        boolean inLeft = dfs(root.left,p,q);
        //在右节点
        boolean inRight = dfs(root.right,p,q);
        if((inLeft && inRight) || (inCurrentNode && (inLeft || inRight))){
            ans = root;
        }
        return inLeft || inRight || inCurrentNode;
    }
}
```
### 第二次碰到
依旧后序遍历，思路和方法二一致。
```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode(int x) { val = x; }
 * }
 */
class Solution {
    TreeNode res = null;
    public TreeNode lowestCommonAncestor(TreeNode root, TreeNode p, TreeNode q) {
        BackOrder(root, p, q);

        return res;
    }
    private boolean BackOrder(TreeNode root, TreeNode p, TreeNode q){
        if(root == null) return false;

        boolean left = BackOrder(root.left, p, q);
        boolean right = BackOrder(root.right, p, q);

        if(left || right){
            if(left && right){
                res = root;
                return true;
            } 

            if(root.val == p.val || root.val == q.val){
                res = root;
                return true;
            }

            return true;
            
        }

        if(root.val == p.val || root.val == q.val){
            return true;
        }

        return false;

    }
}
```
