**画图**，和下标有关系的都要画图确定下标。
还有交换元素时（包括链表操作），尽量多定义变量，这样使得代码更加清晰。
```java
class Solution {

    public void rotate(int[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;

        for (int i = 0; i < m/2; i++) {
            for (int j = i; j < n - i - 1; j++) {
                int x1 = matrix[i][j];
                int x2 = matrix[j][n-i-1];
                int x3 = matrix[n-i-1][n-j-1];
                int x4 = matrix[n-j-1][i];

  
                matrix[i][j] = x4;
                matrix[j][n-i-1] = x1;
                matrix[n-i-1][n-j-1] = x2;
                matrix[n-j-1][i] = x3;
            }
        }
    }
}
```
