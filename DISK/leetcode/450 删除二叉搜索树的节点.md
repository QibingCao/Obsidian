递归查找删除节点，从要删除的结点出发。
找到右子节点的最小值，直接把删除结点的左节点挂在他的左边。

递归的方式，通过返回值链接整个树节点。
```java
TreeNode recurr(TreeNode root, ...){
    if(root == null) return root;

    //root内处理
    。。。
    
    
    //递归
    root.left = recurr(root.left, ...);
    root.right = recurr(root.right, ...);

    return root;
}
```

```java
class Solution {
  public TreeNode deleteNode(TreeNode root, int key) {
    if (root == null) return root;
    if (root.val == key) {
      if (root.left == null) {
        return root.right;
      } else if (root.right == null) {
        return root.left;
      } else {
        TreeNode cur = root.right;
        while (cur.left != null) {
          cur = cur.left;
        }
        cur.left = root.left;
        root = root.right;
        return root;
      }
    }
    if (root.val > key) root.left = deleteNode(root.left, key);
    if (root.val < key) root.right = deleteNode(root.right, key);
    return root;
  }
}
```
