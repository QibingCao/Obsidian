## 层序遍历性质
维护的队列中有如下性质：在每一层元素刚开始出队时，此时队列内的元素均为同一层，只要记住当前队列内元素个数，依次出队，即可保证出队完队列内元素均为下一层。
如果要求答案按照层输出二维List，利用此性质即可。
```java
class Solution {
    List<List<Integer>> res = new LinkedList<>();
    public List<List<Integer>> zigzagLevelOrder(TreeNode root) {
        boolean reverse = false;
        if (root == null) return res;
        Queue<TreeNode> que = new LinkedList<>();
        que.add(root);

        while (!que.isEmpty()) {
            List<Integer> cur = new LinkedList<>();
            int n = que.size(); 【注意】
            for (int i = 0; i < n; i++) {【注意】此处要是que.size()会更新
                TreeNode tmp = que.remove();
                cur.add(tmp.val);
                if (tmp.left != null) que.add(tmp.left);
                if (tmp.right != null) que.add(tmp.right);

                // System.out.println(que.size());
            }
        }
    }
}
```
