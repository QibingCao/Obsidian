原地哈希，在常数空间复杂度下解决。适合寻找某个类似某个特殊自然数的题目。
![[Pasted image 20240330194518.png]]
==第i个位置的元素不是i+1的话，对应下标元素与i位置元素交换==
会在第i个位置一直循环，直到第i个位置符合
```java
// 原地hash，下标i放元素i+1
for (int i = 0; i < n; i++) {
	
	// ==第i个位置的元素不是i+1的话，对应下标元素与i位置元素交换==
	while (前置条件 && nums[i] != i+1) {
		swap(nums, i, nums[i] - 1);
	}
}
```
