https://wdxtub.com/csapp/thick-csapp-lab-1/2016/04/16/

添加32位gcc
``` 
apt install gcc-multilib
```

编译
```
make clean
make btest
```
运行
```
./btest
```

注意：每次更改bits.c文件后都要重新编译btest。如果需要检查单个函数的正确性，可以使用-f标志：
`text $ ./btest -f bitXor`

`dlc`程序可以检测我们有没有违规，如果运行没有输出则没有问题
`text $ ./bits.c`
