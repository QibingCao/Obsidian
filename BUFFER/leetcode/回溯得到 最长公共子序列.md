回溯重建最长公共子序列：
```python
str1(0, m)
str2(0, n)

i = m, j = n
res = ""
while i > 0 and j > 0:
	if str1[i] == str2[j]:
		res += str1[i]
		--i
		--j
	else: // 不相等，那就缩小范围，判断公共子序列来自哪个范围
		if dp[i-1][j] > dp[i][j-1]: // 明确来自i-1，j
			--i
		else：
			--j

reverse(res)
```
